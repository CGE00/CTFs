## Documentación de CTFs

En este espacio recopilo mis notas, soluciones y aprendizajes obtenidos durante la resolución de distintos CTFs.  
Mi objetivo es mejorar mis habilidades en ciberseguridad, análisis de vulnerabilidades y pensamiento crítico, documentando cada reto de forma clara y reproducible.

Aquí encontrarás:
- Metodologías utilizadas
- Pasos de explotación
- Scripts o herramientas aplicadas
- Reflexiones y mejoras para futuros retos
