## Documentación de CTFs

En este repositorio recopilo mis apuntes, soluciones y aprendizajes obtenidos durante la resolución de distintos CTFs.  
Mi objetivo es mejorar mis habilidades en ciberseguridad, análisis de vulnerabilidades y pensamiento crítico, documentando cada reto de la forma mas clara y reproducible que me sea posible.

Contenido principal:
- Metodologías utilizadas
- Pasos de explotación
- Scripts o herramientas aplicadas
- Reflexiones y mejoras para futuros retos
